Programming Languages: Introduction

What is a Programming Language?

A programming language is a system of notation for writing computer programs. It is a kind of computer language that can be text-based or graphical. Programming languages are usually classified into high-level and low-level languages, depending on their level of abstraction from the hardware.

Why Are There So Many Programming Languages Out There?

Every programming language was created to fulfill a different need, for example:

    Different types of applications:

    • Console Applications

    • Graphical User Interface Applications

    Different Platforms:

    • Web Applications

    • Desktop Applications: Windows, Mac, Linux, etc...

    • Mobile Applications: Android and ios
