Why ‘FirstTimeCompile’?

If you've ever written code in a compiled language, you know that every time you write a piece of code and try to run it the compiler will always have some complaint about the code you wrote, and won't let you run it unless you go fix all those compile-time errors. Thus, when you run $ cc main.c, $ g++ main.cpp, or $ cargo run (or simply press F5 in VS Code) and find that your code compiled first time with no errors is the single greatest feeling you can get as a programmer.
