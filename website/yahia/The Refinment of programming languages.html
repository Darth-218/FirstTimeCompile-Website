Programming Languages:
The Refinement


The increased use of high-level languages introfuced a requirement for low-level languages, to a varying degree these languages provide facilities between assembly languages and high-level languages. they can be used to perform tasks that require direct access to the hardware and still provide higher-level control and error-checking.

    APL

    APL intoduced array programming, that allows the application to operate on an entire set of values at once, and influenced functional programming.

    Simula

    In the 1960s, Simula was the first language designed to support object-oriented programming, followed later in the mid 1970s by Smalltalk, the first purely object-oriented language.

    Prolog

    Prolog, in 1972, was the first logic programming language, a program in a logic programming language is a set of sentences in logical form expresssing facts and rules about some problem domain.

Each of these languages has descendants, and most modern programming languages count at least one of them in their ancestry
